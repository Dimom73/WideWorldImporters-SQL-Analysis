# WideWorldImporters SQL Analysis

SQL Server data analysis project based on the Microsoft WideWorldImporters sample database.

This project was completed as part of a Data Analyst course assignment. The goal was to practice SQL data analysis using a real relational database, multiple tables, joins, aggregations, ranking, window functions, and business-oriented reporting.

The original school assignment included several SQL analysis tasks. This repository contains the completed SQL solutions prepared for my project submission. In the course context, it was acceptable to submit the completed tasks rather than publishing every assignment prompt.

## Project Overview

The project uses the WideWorldImporters sample database to answer analytical business questions with SQL.

The analysis focuses on sales performance, customer ranking, product profitability, supplier-product relationships, geographic customer spending, monthly order patterns, churn risk detection, and customer category concentration.

## Database

The project is based on the **WideWorldImporters** sample database for Microsoft SQL Server.

WideWorldImporters is a sample transactional database that includes business entities such as:

- Customers
- Orders
- Order lines
- Invoices
- Invoice lines
- Stock items
- Suppliers
- Cities
- Countries
- Customer categories

## Main Analysis Areas

The SQL file includes queries related to:

- Annual sales and income growth analysis
- Top customers by quarter
- Most profitable products
- Nominal product profit ranking
- Supplier product aggregation
- Top customers by total spending and geographic location
- Monthly order matrix
- Potential customer churn detection
- Customer category business risk analysis
